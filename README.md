# Meridian Android Foundation

Upload this folder to GitHub as the starting Android project.

## Current phase
- Kotlin-only Android app foundation
- Jetpack Compose + Material 3
- Hilt wired
- Room/DataStore dependencies added
- API target 35
- Blank first-run app shell
- No demo/test user data

## Open in Android Studio
1. Open Android Studio
2. File > Open
3. Select this project folder
4. Let Gradle sync
5. Build > Generate Signed Bundle / APK when ready for Play Store testing
